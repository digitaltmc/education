** Installation

#+BEGIN_SRC 
npm install -g @tarojs/cli
# or
yarn global add @tarojs/cli

# Check if taro is installed.
taro -V

taro init myApp

npm run dev:weapp
npm run build:weapp

# Baidu
npm run dev:swan
# Ali
npm run dev:alipay
# H5
npm run dev:h5
# RN
npm run dev:rn

#---------- update taro

taro update self
taro update project

#+END_SRC

*** weapp

添加的路径为项目根目录下的 dist 文件夹。此外，由于 Taro 编译后的代码已经经过了转义和压缩，因此还需要注意微信开发者工具的项目设置：

* 设置关闭 ES6 转 ES5 功能
* 设置关闭上传代码时样式自动补全
* 设置关闭代码压缩上传
